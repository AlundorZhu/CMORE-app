# Installation

## Install git lfs
```
brew install git-lfs
git lfs install
```

## Setup the repo
```
git clone https://github.com/AlundorZhu/CMORE-app.git
cd CMORE-app/
git lfs pull
open CMORE.xcodeproj
```

# Use the app
Press on the record button to start algorithm and recording.
Press again to stop.

The video will be saved to the photo library once you comfirm at the end of recording. The ML results are saved in the Files->On My iPhone->CMORE.
